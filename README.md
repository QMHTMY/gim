### release

### install
* If use deb package  
> $ sudo dpkg -i gim_shell_x.x.x_linux_amd64.deb # shell-script 2.8K
>
> $ sudo dpkg -i gim_golang_x.x.x_linux_amd64.deb # golang 4.4M

* If use src code  
> $ cd shell/ or cd golang/
>
> $ make   
>
> $ make install

### uninstall
* uninstall deb
> $ sudo dpkg -r gim
* uninstall src
> $ make uninstall
